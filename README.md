# Cronômetro em C (Display com Caracteres)

Este projeto consiste no desenvolvimento de um cronômetro de três dígitos exibido na tela utilizando caracteres, implementado na linguagem C. O programa simula um display numérico simples e permite ao usuário controlar o tempo através do teclado, além de salvar configurações e estado do cronômetro em arquivo.

O trabalho foi desenvolvido como atividade da disciplina de Elementos de Programação do curso técnico em Eletrônica da Fundação Liberato.
